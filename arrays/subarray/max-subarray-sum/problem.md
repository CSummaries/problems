Given an array of integers (positive and negative), write a function that will return a subarray with the maximum sum.

# Examples

* Input: `[-40, -10, -2, 1]`<br/>
  Output: `[1]`
* Input: `[100, -99, 1, 2, 3]`<br/>
  Output: `[100]`
* Input: `[1, 2, 3, 4, 5, -1]`<br/>
  Output: `[1, 2, 3, 4, 5]`
